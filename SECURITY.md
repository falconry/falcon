# Security Policy

## Supported Versions

Falcon actively supports the current stable release, and provides limited
security-only patches for the latest minor release of the previous stable
major series. See
[Releases and Versioning](https://falcon.readthedocs.io/en/stable/community/releases.html#supported-releases)
for the full policy, including how reported vulnerabilities are evaluated.

## Reporting a Vulnerability

Please **do not** report security vulnerabilities through public GitHub issues.

Instead, please use GitHub's private vulnerability reporting feature:

1. Go to the [Security tab](https://github.com/falconry/falcon/security) of this repository.
2. Click **Report a vulnerability**.
3. Provide as much detail as possible, including steps to reproduce, affected
   versions, and potential impact.

You should receive an acknowledgement within a few days. If you haven't heard
back, please follow up via the same advisory thread.

Non-sensitive security questions (e.g., general hardening advice) are welcome
on the [Falcon Gitter channel](https://gitter.im/falconry/dev) or the
[users mailing list](https://mail.falconframework.org/archives/list/users@mail.falconframework.org/).
