Changelog for Falcon 4.4.0
==========================

Summary
-------

Falcon 4.4 is in development. The progress is tracked via the
`Version 4.4 milestone <https://github.com/falconry/falcon/milestone/46>`__
on GitHub.


Changes to Supported Platforms
------------------------------

- CPython 3.15 is now fully supported.
  (`#2593 <https://github.com/falconry/falcon/issues/2593>`__)
