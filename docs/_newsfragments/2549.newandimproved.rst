Added a new :meth:`~falcon.Request.get_param_as_media` method to
:class:`~falcon.Request` and :class:`~falcon.asgi.Request` that
deserializes a single query-string parameter using the configured media
handlers. The helper accepts an optional ``media_type`` (which falls back
to the app's ``default_media_type``), can store the parsed value into a
provided mapping, and raises ``HTTPInvalidParam`` on parse errors.
